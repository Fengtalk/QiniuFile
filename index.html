Qiniufile

在 Typecho 中使用七牛作为存储

View on GitHub Download .zip Download .tar.gz

写了这么多年博客，很容易就把博客的配图给弄丢。因此，我需要一个图床，将这些图片可靠地保存起来。使用七牛有相当长的一段时间，这个时候自然想到它。

七牛是个不错的云储存平台，注册成为标准用户之后，享有以下权利：

  1. 免费存储空间10GB
  2. 免费每月下载流量10GB
  3. 免费每月PUT/DELETE 10万次请求
  4. 免费每月GET 100万次请求

把 Typecho 作为博客平台之后，一直在寻求七牛的相关的插件。无可奈何的是，之前在网上找的都不能正常使用。大家都那么忙，你都不好意思让人家更新。

最后在论坛找到了这么一款插件，针对之前 QiniuFile 做了修改，能够在目前最新版的 Typecho 1.0 上正常使用。

在此对原作者和修改者表示感谢。下面附上插件的地址。

https://github.com/Fengtalk/qiniu-typecho-file
